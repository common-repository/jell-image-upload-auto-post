== Description ==

`Creates posts automatically using media gallery` 
This plugin will let you create single/bulk post after uploading any media from wordpress media gallery. This is very much essential plugin for photo blog or where there are a lots number of image posting in a wordpress driven site.

When you will upload an image from wordpress media gallery then a post will be created automatically with that image as featured image.

`Note:` Post title, image title, caption, alt etc will be image file name. So change image file name before upload.

`How it works:`

1. login to your wordpress site
1. Click on Media Menu and upload images
1. This plugin create automatically posts of those images

`Supported Features:`

* Automatically Create post of Image (including post title, description, Category)
* Automatically Post Category
* Automatically Post tags
* Automatically Post Featured Image
* Automatically image title, alt, caption, description
* Automatically Media Type (like photo, Vector etc)
* Automatically Image License (like Public Domain Work etc)
* Easy backend options to control automated posts 
* Post template format
* Available many Template Tags , Short tags

== Installation ==

To **Install** this `Jell Image Upload Auto Post` plugin there is no any complexity. It's very simple like other plugin. Just follow the procedure described below.

1. Download the plugin from Wordpress Plugin repository. After downloading the zip file extract it.
2. Upload `jell-image-upload-auto-post` plugin directory to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Upload an image from 'Add Media' menu in Wordpress. Then go to 'All Post' menu and see after uploading that image there will be a new post with that image.
